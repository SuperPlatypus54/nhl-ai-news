## NHL AI News — local setup

1. Copy the example env file and add your OpenAI key:

```bash
cp .env.local.example .env.local
# then edit .env.local and put your real OPENAI_API_KEY
```

2. Install dependencies (already done by create-next-app):

```bash
npm install
```

3. Run the dev server:

```bash
npm run dev
```

4. Open http://localhost:3000 and try typing an NHL headline and clicking "Generate Story".

## Deploying to Vercel

When you deploy on Vercel, set the environment variable `OPENAI_API_KEY` in the Vercel dashboard (Project Settings → Environment Variables). Optionally set `OPENAI_MODEL` if you want to override the default.
